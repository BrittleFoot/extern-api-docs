Примеры
=======

Разберем несколько сценариев работы с API Контур.Экстерна, подробно, с примерами запросов и ответов на них.

Если вы хотите сами поработать с API в тестовой среде, то вам необходимо получить тестовые данные. Для этого необходимо написать нам по адресу extern-api@skbkontur.ru. В письме необходимо указать наименование и ИНН вашей организации, кратко описать суть вашей деятельности и планируемые сценарии использования API Контур.Экстерна.

.. important:: В примерах используются готовые файла документов, при использовании их в своих запросах, необходимо указать в них актуальные для вашей тестовой учетной записи данные.
